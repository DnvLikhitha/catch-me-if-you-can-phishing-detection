# PhishGuard AI - Extension Icons

## Required Icons
The extension needs icon files in the following sizes:
- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

## Icon Design Guidelines
- Use shield or security-themed imagery
- Primary colors: Blue (#3b82f6) and Purple (#764ba2)
- Maintain high contrast for visibility
- Include recognizable security symbolism

## Temporary Placeholder
Until proper icons are created, you can:
1. Use online icon generators (e.g., favicon.io)
2. Design with tools like Figma, Canva, or Adobe Illustrator
3. Use a shield emoji as a base and create PNG versions

## Quick Generation
Visit https://favicon.io/emoji-favicons/shield/ and generate shield emoji icons.

After generating, place the PNG files in this directory:
- extension/icons/icon16.png
- extension/icons/icon32.png
- extension/icons/icon48.png
- extension/icons/icon128.png
